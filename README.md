# SoC TopGen UI

System on Chip (SoC) トポロジー生成のためのビジュアルUIツール

## 概要

SoC TopGen UIは、SoCアーキテクチャを視覚的に設計し、トポロジー定義を生成するためのWebベースのアプリケーションです。ドラッグ&ドロップによる直感的なインターフェースで、複雑なSoC設計を簡単に行うことができます。

## 主要機能

- 🎨 **ビジュアルエディタ**: ドラッグ&ドロップでコンポーネントを配置・接続
- 📦 **コンポーネントライブラリ**: CPU、メモリ、インターコネクト、ペリフェラルなど
- ✅ **バリデーション**: トポロジーの整合性チェック
- 💾 **プロジェクト管理**: 設計の保存・読み込み
- 🚀 **コード生成**: SystemVerilog、YAML、ドキュメントのエクスポート

## プロジェクト構造

```
soc-topgen-ui/
├── src/
│   ├── components/     # 再利用可能なUIコンポーネント
│   ├── pages/          # ページコンポーネント
│   ├── hooks/          # カスタムReactフック
│   ├── services/       # ビジネスロジック、API
│   ├── types/          # TypeScript型定義
│   └── utils/          # ユーティリティ関数
├── docs/               # ドキュメント
│   ├── SPEC.md         # 仕様書
│   └── ROADMAP.md      # 開発ロードマップ
├── tests/              # テストファイル
├── config/             # 設定ファイル
└── public/             # 静的ファイル
```

## 技術スタック

- **フロントエンド**: React 18 + TypeScript
- **状態管理**: Redux Toolkit / Zustand
- **UIライブラリ**: Material-UI / Ant Design
- **グラフィックス**: React Flow
- **ビルドツール**: Vite
- **テスト**: Jest + React Testing Library

## セットアップ
